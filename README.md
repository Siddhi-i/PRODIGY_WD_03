# PRODIGY_WD_03

## Task: Tic-Tac-Toe Web Application

This project is a **Tic-Tac-Toe Web Application** developed as part of the **Web Development Internship at Prodigy InfoTech**.

The game allows two players to play against each other, tracks the game state, checks for winning conditions, and displays the result.

---

## Features
- Interactive 3Ã—3 Tic-Tac-Toe board
- Alternating turns for Player X and Player O
- Automatic win and draw detection
- Reset game functionality
- Simple and user-friendly interface

---

## Technologies Used
- HTML
- CSS
- JavaScript

---

## How to Run the Project
1. Download or clone the repository
2. Open the `index.html` file in any web browser
3. Click on the cells to start playing
4. Use the reset button to restart the game
